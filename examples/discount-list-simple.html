👋 Welcome!

This page demonstrates the data-next-discount-list enhancer. The discount list will automatically display:

  • Offer discounts - Applied automatically based on campaign offers
  • Voucher discounts - Applied via coupon codes

Note: This requires actual offer/voucher configuration in your NextCommerce campaign.

📖 Quick Reference

Attributes:

  • data-next-discount-list - Required identifier
  • data-discount-template - Custom inline template (optional)
  • data-discount-template-id - ID of template element (optional)
  • data-empty-template - Content when no discounts (optional)
  • data-show-item-discounts - Include item-level discounts (optional)
  • data-group-by-offer - Group by offer ID, default true (optional)

Template Variables:

  • {discount.name} - Discount name
  • {discount.description} - Description
  • {discount.amount} - Formatted amount (e.g., "$15.99")
  • {discount.amount.raw} - Raw number (e.g., 15.99)
  • {discount.type} - "offer" or "voucher"
  • {discount.offerId} - Offer ID if applicable
  • {discount.showDescription} - "show" or "hide" class

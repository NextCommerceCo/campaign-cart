### Simple Add to Cart

The most basic implementation - a product with an add-to-cart button and cart display.

  <iframe

**Key Features Demonstrated:**
- `data-next-action="add-to-cart"` - Add items to cart
- `data-next-display="cart.total"` - Show cart total
- `data-next-display="cart.itemCount"` - Display item count
- Automatic cart persistence

## Code Snippets Library

Quick copy-paste snippets for common functionality:

<Tabs>
  <TabItem label="Add to Cart Button">
    <Code code={`<button data-next-action="add-to-cart" 
        data-next-package-id="your-product-id"
        data-next-quantity="1">
  Add to Cart
</button>`} lang="html" />
  </TabItem>
  
  <TabItem label="Cart Total Display">
    <Code code={`<div class="cart-summary">
  Items: <span data-next-display="cart.itemCount">0</span>
  Total: <span data-next-display="cart.total">$0.00</span>
</div>`} lang="html" />
  </TabItem>
  
  <TabItem label="Product Selector">
    <Code code={`<div data-next-cart-selector data-next-selection-mode="swap">
  <div data-next-selector-card data-next-package-id="basic">
    <h3>Basic Plan</h3>
    <p data-next-display="package.price">$19/mo</p>
  </div>
  <div data-next-selector-card data-next-package-id="pro">
    <h3>Pro Plan</h3>
    <p data-next-display="package.price">$39/mo</p>
  </div>
</div>`} lang="html" />
  </TabItem>
  
  <TabItem label="Conditional Display">
    <Code code={`<!-- Show when cart has items -->
<div data-next-show="cart.hasItems">
  <button onclick="checkout()">Checkout</button>
</div>

<!-- Show when cart is empty -->
<div data-next-show="cart.isEmpty">
  <p>Your cart is empty</p>
</div>`} lang="html" />
  </TabItem>
  
  <TabItem label="Quantity Controls">
    <Code code={`<div data-next-quantity-controls data-next-package-id="product-id">
  <button data-next-quantity-decrease>-</button>
  <input data-next-quantity-input type="number" value="1" min="1">
  <button data-next-quantity-increase>+</button>
</div>`} lang="html" />
  </TabItem>
</Tabs>

## Interactive Debugging

Test your implementations with our built-in debugging tools:

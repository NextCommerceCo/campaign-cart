## Installation Steps

Now let's set this up on your site:

<Steps>

1. **Get your API key**

   Sign up at [Campaign Cart Dashboard](https://dashboard.campaigncart.com) and copy your API key from settings.

2. **Add to your HTML**

   Add this to your `<head>` tag:

   <SDKSetup />

3. **Add your first button**

   Add a button anywhere on your page:

   ```html
   <button data-next-action="add-to-cart" data-next-package-id="1">
     Add to Cart - $99
   </button>
   ```

</Steps>

## Core Concepts

### Data Attributes

Campaign Cart uses HTML data attributes to add functionality:

| Attribute | Purpose | Example |
|-----------|---------|---------|
| `data-next-action` | Triggers actions | `"add-to-cart"`, `"checkout"`, `"clear-cart"` |
| `data-next-display` | Shows dynamic data | `"cart.total"`, `"package.price"` |
| `data-next-show` | Conditional visibility | `"cart.hasItems"`, `"cart.isEmpty"` |
| `data-next-package-id` | Product identifier | `"1"`, `"2"`, `"selected"` |

### Basic Configuration

The simplest configuration only needs an API key:

```javascript
window.nextConfig = {
  apiKey: "your-api-key-here"
};
```

For more options, see the [Configuration Reference](/config/configuration-reference/).

## Testing Your Integration

1. **Open Browser Console** - Look for "Campaign Cart SDK initialized"
2. **Click Add to Cart** - Watch the cart count update
3. **Try Checkout** - Should redirect to checkout page
4. **Check Network Tab** - Verify API calls are working

<Aside type="caution">
**Checkout not working?** Make sure you've set up products in your Campaign Cart dashboard first.
</Aside>

## Frequently Asked Questions

**Q: Do I need a backend server?**  
No! Campaign Cart runs entirely in the browser. Just add the SDK to your HTML.

**Q: How do I create products?**  
Products are created and managed in your [Campaign Cart dashboard](https://dashboard.campaigncart.com).

**Q: Can I customize the checkout page?**  
Yes! The checkout page can be fully customized in your dashboard settings.

**Q: Is it secure?**  
Yes. All payment processing happens on PCI-compliant servers. Your site never handles sensitive payment data.

**Q: How do I test without real payments?**  
Use test mode in your dashboard. You can also use our demo API key for initial testing.

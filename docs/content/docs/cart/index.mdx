---
title: Cart System
description: Enhancers for package selection, adding items, quantity controls, and rendering cart contents.
---

The cart system provides everything needed to let users select packages, manage quantities, and proceed to checkout.

## Enhancers in this section

| Enhancer | Attribute | Purpose |
|---|---|---|
| [`PackageSelectorEnhancer`](/docs/cart/package-selector) | `data-next-cart-selector` | Group of selectable package option cards |
| [`AddToCartEnhancer`](/docs/cart/add-to-cart) | `data-next-action="add-to-cart"` | Button that adds a package to the cart |
| [`CartToggleEnhancer`](/docs/cart/cart-toggle) | `data-next-toggle` | Toggle button — adds when not in cart, removes when in cart |
| [`CartItemListEnhancer`](/docs/cart/cart-items) | `data-next-cart-items` | Renders the list of cart items from a template |
| [`QuantityControlEnhancer`](/docs/cart/quantity-controls) | `data-next-quantity` | Increase / decrease / set item quantity |
| `RemoveItemEnhancer` | `data-next-remove-item` | Button to remove a specific item from the cart |
| [`CartOfferListEnhancer`](/docs/cart/cart-offer-list) | `data-next-offer-list` | Renders campaign offers as selectable cards and applies them to the cart |
| [`CartItemSlotsEnhancer`](/docs/cart/cart-item-slots) | `data-next-cart-slots` | Expands cart items into individual per-unit slot cards with variant selectors |

---

## Common patterns

### Select + Buy (most common)

Use a selector in `select` mode paired with an `AddToCartEnhancer` button:

```html
<div data-next-cart-selector data-next-selector-id="main" data-next-selection-mode="select">
  <div data-next-selector-card data-next-package-id="10" data-next-selected="true">1 Bottle</div>
  <div data-next-selector-card data-next-package-id="20">3 Bottles</div>
</div>

<button data-next-action="add-to-cart" data-next-selector-id="main" data-next-url="/checkout">
  Buy Now
</button>
```

### Direct add

Skip the selector and add a specific package directly:

```html
<button data-next-action="add-to-cart" data-next-package-id="10" data-next-url="/checkout">
  Buy Now
</button>
```

### Order bump

An optional add-on item with a toggle button:

```html
<div data-next-bump>
  <button data-next-toggle
          data-next-package-id="99"
          data-add-text="Yes, add protection plan"
          data-remove-text="Protection plan added ✓">
    Yes, add protection plan
  </button>
</div>
```

### Offer selection with per-unit variant configuration

Use `CartOfferListEnhancer` to let users pick an offer, then `CartItemSlotsEnhancer` to configure each unit:

```html
<div data-next-offer-list data-next-offer-template-id="offer-tpl"></div>

<script id="offer-tpl" type="text/html">
  <div class="offer-card" data-next-offer-card
       data-next-offer-id="{offer.ref_id}"
       data-next-offer-package-id="{offer.package.id}">
    {offer.name} — {offer.package.price}
  </div>
</script>

<div data-next-cart-slots data-next-slot-template-id="slot-tpl"></div>

<template id="slot-tpl">
  <div class="slot-card">
    <span>Unit #{slot.index}</span>
    <p>{item.variantName}</p>
    <div data-next-variant-selectors></div>
  </div>
</template>
```

### Cart summary with item list

```html
<div data-next-cart-items data-item-template-id="cart-row-tmpl">
  <!-- items rendered here -->
</div>

<script id="cart-row-tmpl" type="text/html">
  <div class="cart-row">
    {item.name} × {item.quantity}
    <span>{item.finalLineTotal}</span>
    <button data-next-remove-item data-package-id="{item.packageId}">Remove</button>
  </div>
</script>

<p>Total: <span data-next-display="cart.total"></span></p>
```

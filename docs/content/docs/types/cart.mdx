## CartItem

A single item in the cart, enriched with package and product data.

<AutoTypeTable path="../src/types/global.ts" name="CartItem" />

### `packageId`

**Type:** `number`

The campaign package `ref_id` for this item. Use this to look up package data or match items to selector elements.

```ts
const item = NextCommerce.getCart().items.find(i => i.packageId === 101);
```

### `quantity` vs `qty`

`quantity` is the number of packages in the cart (how many times the customer added it). `qty` is the number of product units per package (set by the campaign).

```ts
const item = cart.items[0];
const totalUnits = item.quantity * (item.qty ?? 1);
// quantity=2 packages × qty=3 units = 6 total units
```

### `price` vs `price_total`

`price` is a raw number used for calculations. `price_total` is the formatted string from the campaign API for display.

```ts
const item = cart.items[0];
console.log(item.price);       // 4997
console.log(item.price_total); // "$49.97"
```

### `is_upsell`

`true` when the item was added via a post-purchase upsell page, not a standard add-to-cart action. Use this to differentiate upsell items in analytics or display logic.

### `is_recurring`

`true` for subscription products. Pair with `interval` and `interval_count` to display billing frequency.

```ts
if (item.is_recurring) {
  // "Billed every 1 month"
  console.log(`Billed every ${item.interval_count} ${item.interval}`);
}
```

### Discount fields

When an offer is active, discounted prices are available alongside the original prices:

| Field | Description |
| ----- | ----------- |
| `unit_price_incl_discount` | Per-unit price after discount, tax-inclusive |
| `unit_price_excl_discount` | Per-unit price after discount, tax-exclusive |
| `package_price_incl_discount` | Total package price after discount, tax-inclusive |
| `package_price_excl_discount` | Total package price after discount, tax-exclusive |
| `total_discount` | Total discount amount for this line |

## CartState

The full cart state managed by the cart store.

<AutoTypeTable path="../src/types/global.ts" name="CartState" />

### Accessing cart state

```ts
// One-time read
const cart = NextCommerce.getCart();

// Subscribe to changes
NextCommerce.on('cart:updated', (cart) => {
  console.log(cart.totals.total.formatted); // "$149.97"
});
```

### `items` vs `enrichedItems`

`items` is the raw cart array — lightweight, used for add/remove operations. `enrichedItems` is the same data enriched with structured pricing breakdowns for display rendering.

```ts
// Use items for cart operations
cart.items.forEach(item => {
  console.log(item.packageId, item.quantity);
});

// Use enrichedItems for display
cart.enrichedItems.forEach(line => {
  console.log(line.price.incl_tax.formatted); // "$49.97"
  console.log(line.price.savings.formatted);  // "$10.00"
});
```

### `totals`

The `totals` object is the primary source for displaying cart amounts. Every field includes both a raw `value` and a currency-formatted `formatted` string.

```ts
const { totals } = NextCommerce.getCart();

console.log(totals.subtotal.formatted);    // "$149.97"
console.log(totals.shipping.formatted);    // "$9.99"
console.log(totals.discounts.formatted);   // "$30.00"
console.log(totals.total.formatted);       // "$129.96"
console.log(totals.hasSavings);            // true
console.log(totals.savings.formatted);     // "$30.00"
```

### `appliedCoupons`

```ts
const { appliedCoupons } = NextCommerce.getCart();

appliedCoupons.forEach(coupon => {
  console.log(coupon.code);     // "SAVE20"
  console.log(coupon.discount); // 29.99 (calculated amount)
});
```

## CartTotals

Computed totals for the cart. Every monetary field has both a raw `value` number and a currency-formatted `formatted` string.

<AutoTypeTable path="../src/types/global.ts" name="CartTotals" />

### Savings display pattern

Use `hasSavings` before rendering savings UI to avoid showing `"You save $0.00"`:

```ts
const { totals } = NextCommerce.getCart();

if (totals.hasSavings) {
  // Show: "You save $30.00 (20% off)"
  showSavings(totals.savings.formatted, totals.savingsPercentage.formatted);
}

if (totals.hasTotalSavings) {
  // Total savings including offer discounts
  showTotalSavings(totals.totalSavings.formatted);
}
```

### Free shipping threshold

Use `totalExclShipping` to calculate how far a customer is from free shipping:

```ts
const { totals } = NextCommerce.getCart();
const freeShippingThreshold = 100;
const amountNeeded = freeShippingThreshold - totals.totalExclShipping.value;

if (amountNeeded > 0) {
  showMessage(`Add $${amountNeeded.toFixed(2)} more for free shipping`);
}
```

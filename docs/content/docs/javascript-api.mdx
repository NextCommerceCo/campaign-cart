---
title: JavaScript API
description: Programmatic access to the SDK via window.NextCommerce. Use when data attributes aren't enough.
---

The `window.NextCommerce` object exposes the full SDK programmatically. This is useful for:

- Custom event handling
- Dynamic cart manipulation
- A/B testing and analytics integration
- Building custom UI components

All methods are available after the SDK is initialized:

```js
window.nextReady = window.nextReady || [];
window.nextReady.push(function(sdk) {
  // SDK is ready
  console.log(sdk.getCartCount()); // 0
});
```

Or access directly after page load:

```js
const sdk = window.NextCommerce;
```

---

## Cart methods

### `addItem(options)`

Add a package to the cart.

```js
await sdk.addItem({ packageId: 42 });
await sdk.addItem({ packageId: 42, quantity: 3 });
```

### `removeItem(options)`

Remove a package from the cart.

```js
await sdk.removeItem({ packageId: 42 });
```

### `updateQuantity(options)`

Set the quantity of a cart item.

```js
await sdk.updateQuantity({ packageId: 42, quantity: 2 });
```

### `clearCart()`

Remove all items from the cart.

```js
await sdk.clearCart();
```

### `swapCart(items)`

Replace the entire cart contents with a new set of items.

```js
await sdk.swapCart([
  { packageId: 10, quantity: 1 },
  { packageId: 99, quantity: 2 }
]);
```

### `hasItemInCart(options)`

Check whether a package is in the cart.

```js
if (sdk.hasItemInCart({ packageId: 42 })) {
  console.log('Package 42 is in the cart');
}
```

## Cart data

### `getCartCount()`

Total number of units across all cart items.

```js
const count = sdk.getCartCount(); // e.g. 3
```

### `getCartTotals()`

Full cart totals object.

```js
const totals = sdk.getCartTotals();
// totals.total.formatted  → "$49.00"
// totals.total.amount     → 49
// totals.savings.formatted → "$20.00"
```

### `getCartData()`

Full cart data including enriched items and campaign data.

```js
const data = sdk.getCartData();
// data.cartLines         → array of enriched cart items
// data.cartTotals        → totals object
// data.campaignData      → campaign object
// data.appliedCoupons    → array of applied coupons
```

## Campaign / package data

### `getCampaignData()`

Returns the full campaign object.

```js
const campaign = sdk.getCampaignData();
// campaign.name, campaign.currency, campaign.packages, etc.
```

### `getPackage(id)`

Returns a single package by ID.

```js
const pkg = sdk.getPackage(42);
// pkg.name, pkg.price, pkg.qty, etc.
```

## Coupon methods

### `applyCoupon(code)`

Apply a coupon code to the cart.

```js
const result = await sdk.applyCoupon('SAVE10');
// result.success → true | false
// result.message → "Coupon applied" | "Invalid coupon"
```

### `removeCoupon(code)`

Remove a specific coupon.

```js
sdk.removeCoupon('SAVE10');
```

### `getCoupons()`

Returns the list of currently applied coupons.

```js
const coupons = sdk.getCoupons();
// [{ code: "SAVE10", discountAmount: 5, type: "percentage", value: 10 }]
```

## Shipping methods

### `getShippingMethods()`

Returns available shipping methods from the campaign.

```js
const methods = sdk.getShippingMethods();
// [{ ref_id: 1, code: "standard", price: "4.99" }, …]
```

### `setShippingMethod(methodId)`

Select a shipping method.

```js
await sdk.setShippingMethod(1);
```

### `getSelectedShippingMethod()`

Returns the currently selected shipping method.

```js
const method = sdk.getSelectedShippingMethod();
// { id: 1, name: "Standard Shipping", price: 4.99, code: "standard" }
```

## Events

### `on(event, handler)`

Listen to an SDK event.

```js
sdk.on('cart:item-added', ({ packageId, quantity }) => {
  console.log('Added', quantity, 'of package', packageId);
});

sdk.on('selector:selection-changed', ({ selectorId, packageId }) => {
  console.log('Selection changed in', selectorId, '→', packageId);
});
```

### `off(event, handler)`

Remove an event listener.

```js
const handler = ({ packageId }) => console.log(packageId);
sdk.on('cart:item-added', handler);
sdk.off('cart:item-added', handler);
```

### Available events

| Event | Payload |
|---|---|
| `cart:item-added` | `{ packageId, quantity }` |
| `cart:item-removed` | `{ packageId }` |
| `cart:quantity-changed` | `{ packageId, quantity }` |
| `selector:item-selected` | `{ selectorId, packageId, previousPackageId, mode }` |
| `selector:selection-changed` | `{ selectorId, packageId, item }` |
| `selector:quantity-changed` | `{ selectorId, packageId, quantity }` |
| `variant:option-selected` | `{ productId, attributeCode, value }` |
| `variant:package-changed` | `{ productId, packageId, variantKey }` |
| `upsell:added` | `{ packageId, quantity, order, value }` |
| `upsell:accepted` | `{ packageId }` |
| `timer:expired` | `{}` |
| `currency:fallback` | `{ requested, fallback }` |

## Post-purchase upsells

### `addUpsell(options)`

Add items to an existing completed order (post-purchase upsell).

```js
// Single item
const result = await sdk.addUpsell({ packageId: 55, quantity: 1 });

// Multiple items
const result = await sdk.addUpsell({
  items: [
    { packageId: 55, quantity: 1 },
    { packageId: 60, quantity: 2 }
  ]
});

// result.order        → updated order object
// result.addedLines   → new order lines
// result.totalValue   → total value of added items
```

### `canAddUpsells()`

Check whether the current order supports upsells.

```js
if (sdk.canAddUpsells()) {
  // Show upsell offer
}
```

### `isUpsellAlreadyAdded(packageId)`

Check whether a upsell package was already accepted.

```js
if (!sdk.isUpsellAlreadyAdded(55)) {
  // Show the offer
}
```

## Analytics

Manual analytics tracking (in addition to automatic tracking):

```js
// Product views
await sdk.trackViewItem(42);
await sdk.trackViewItemList([10, 20, 30], undefined, 'homepage');

// Cart events
await sdk.trackAddToCart(42, 1);
await sdk.trackRemoveFromCart(42, 1);
await sdk.trackBeginCheckout();

// Conversion
await sdk.trackPurchase(orderData);

// Custom event
await sdk.trackCustomEvent('quiz_completed', { score: 3 });
```

## Profiles

Pricing profiles let you apply different package mappings and prices for different customer segments:

```js
// Register a profile
sdk.registerProfile({
  id: 'wholesale',
  name: 'Wholesale',
  packageMappings: { 10: 110, 20: 120 }  // originalId → mappedId
});

// Activate a profile
await sdk.setProfile('wholesale');

// Revert to default
await sdk.revertProfile();

// Query
sdk.getActiveProfile();           // "wholesale" | null
sdk.hasProfile('wholesale');      // true
sdk.listProfiles();               // ["wholesale", "vip"]
```

## Attribution metadata

Attach custom metadata to orders:

```js
// Add a single key
sdk.addMetadata('funnel_step', 'checkout');

// Set multiple keys at once
sdk.setMetadata({ affiliate_id: 'ABC123', quiz_score: 3 });

// Read metadata
const meta = sdk.getMetadata();
```

## URL parameters

Programmatically read or set URL parameters that affect SDK behavior:

```js
// Read
const source = sdk.getParam('utm_source');  // "facebook" | null
sdk.hasParam('gclid');                       // true | false

// Write
sdk.setParam('currency', 'EUR');
sdk.setParams({ profile: 'vip', currency: 'EUR' });
sdk.mergeParams({ custom_param: 'value' });

// Clear
sdk.clearParam('preview');
sdk.clearAllParams();
```

## Utility methods

### `formatPrice(amount, currency?)`

Format a number as currency using the campaign's locale settings.

```js
sdk.formatPrice(49.99);          // "$49.99"
sdk.formatPrice(49.99, 'EUR');   // "€49.99"
```

### `getVersion()`

Returns the SDK version string.

```js
sdk.getVersion(); // "2.3.1"
```

### `validateCheckout()`

Check whether the cart is ready for checkout.

```js
const { valid, errors } = sdk.validateCheckout();
if (!valid) {
  alert(errors.join('\n'));
}
```

## Exit intent

Configure an exit intent popup:

```js
await sdk.exitIntent({
  image: '/popup.png',
  action: async () => {
    await sdk.applyCoupon('STAYFORMORE');
    window.location.href = '/checkout';
  },
  maxTriggers: 1,
  useSessionStorage: true,
  overlayClosable: true
});
```

### `disableExitIntent()`

```js
sdk.disableExitIntent();
```

## FOMO popup

Show a social proof popup with recent activity:

```js
await sdk.fomo({
  items: [
    { text: 'Sarah from New York just ordered!', image: '/avatar1.png' },
    { text: '23 people are viewing this page', image: '/fire.png' }
  ],
  displayDuration: 4000,   // ms to show each popup
  delayBetween: 8000,      // ms between popups
  initialDelay: 5000,      // ms before first popup
  maxMobileShows: 2
});

sdk.stopFomo();
```

---
title: Cart Display
description: Show cart totals, item count, and state. Updates live as items are added or removed.
---

`CartDisplayEnhancer` handles all `data-next-display="cart.*"` elements. It subscribes to the cart store and updates the element's text content whenever the cart changes.

---

## Usage

```html
<span data-next-display="cart.total"></span>
<span data-next-display="cart.itemCount"></span>
<span data-next-display="cart.savings"></span>
```

---

## Available properties

### Totals

| Property | Description | Example |
|---|---|---|
| `cart.total` | Cart total | `$49.00` |
| `cart.subtotal` | Subtotal before shipping and tax | `$49.00` |
| `cart.shipping` | Shipping cost | `$4.99` |
| `cart.tax` | Tax amount | `$3.92` |
| `cart.discounts` | Total coupon discounts | `$10.00` |
| `cart.savings` | Savings vs retail prices | `$20.00` |
| `cart.savingsPercentage` | Savings as a percentage | `29%` |
| `cart.compareTotal` | Original retail total | `$69.00` |

### Counts

| Property | Description | Example |
|---|---|---|
| `cart.itemCount` | Number of distinct line items | `2` |
| `cart.totalQuantity` | Total units across all items | `4` |

### State flags

| Property | Description | Returns |
|---|---|---|
| `cart.hasItems` | Cart is not empty | `"true"` / `"false"` |
| `cart.isEmpty` | Cart is empty | `"true"` / `"false"` |

### Currency

| Property | Description | Example |
|---|---|---|
| `cart.currency` / `cart.currencyCode` | Currency code | `USD` |
| `cart.currencySymbol` | Currency symbol | `$` |

---

## Raw values

Append `.raw` to any numeric property to get an unformatted number:

```html
<!-- Formatted: "$49.00" -->
<span data-next-display="cart.total"></span>

<!-- Raw: "49" -->
<span data-next-display="cart.total.raw"></span>
```

---

## Subtotal with discounts

Add `data-include-discounts` to a `subtotal` element to subtract applied coupon discounts:

```html
<span data-next-display="cart.subtotal" data-include-discounts></span>
```

---

## CSS classes

The SDK automatically toggles these classes on every `cart.*` display element:

| Class | When |
|---|---|
| `next-cart-empty` | Cart is empty |
| `next-cart-has-items` | Cart has at least one item |

Use these to show/hide elements based on cart state:

```css
.cart-summary {
  display: none;
}

.cart-summary.next-cart-has-items {
  display: block;
}
```

## Examples

### Cart badge (item count)

```html
<div class="cart-icon">
  🛒 <span data-next-display="cart.totalQuantity"></span>
</div>
```

### Savings callout

```html
<div class="savings-banner" data-next-show="cart.hasSavings">
  You're saving <strong data-next-display="cart.savings"></strong>
  (<span data-next-display="cart.savingsPercentage"></span> off retail)
</div>
```

### Order total with breakdown

```html
<table>
  <tr>
    <td>Subtotal</td>
    <td data-next-display="cart.subtotal"></td>
  </tr>
  <tr>
    <td>Shipping</td>
    <td data-next-display="cart.shipping"></td>
  </tr>
  <tr>
    <td>Tax</td>
    <td data-next-display="cart.tax"></td>
  </tr>
  <tr>
    <td><strong>Total</strong></td>
    <td><strong data-next-display="cart.total"></strong></td>
  </tr>
</table>
```

---
title: Product Display
description: Display package prices, savings, and product data. Resolves package context from the DOM.
---

`ProductDisplayEnhancer` handles `data-next-display="package.*"` elements. It reads package data from the campaign store and re-renders when campaign data or pricing changes (e.g. when a coupon is applied or the currency changes).

---

## Package context

The enhancer needs to know **which package** to display. It resolves this in order:

1. **Explicit ID in path**: `data-next-display="package.10.price"` — uses package `10`
2. **Ancestor with `data-next-package-id`**: walks up the DOM to find a container with `data-next-package-id="10"`
3. **Selected item**: resolves via the `PackageContextResolver` (reads selector state)

---

## Usage

### Explicit package ID in path

```html
<span data-next-display="package.10.price"></span>
<span data-next-display="package.10.price_retail"></span>
```

### Ancestor context (preferred for cards)

```html
<div data-next-package-id="10">
  <span data-next-display="package.name"></span>
  <del data-next-display="package.price_retail"></del>
  <strong data-next-display="package.price"></strong>
  <small>Save <span data-next-display="package.savingsPercentage"></span></small>
</div>
```

## Available properties

### Basic info

| Property | Description |
|---|---|
| `package.name` | Package name |
| `package.image` | Product image URL |
| `package.qty` | Units per package |
| `package.is_recurring` | `"true"` if subscription |

### Pricing

| Property | Description |
|---|---|
| `package.price` | Sale price (package total) |
| `package.price_total` | Sale price × qty |
| `package.price_retail` | Retail/compare price |
| `package.price_retail_total` | Retail price × qty |
| `package.price_recurring` | Recurring charge price |
| `package.unitPrice` | Per-unit sale price |
| `package.unitRetailPrice` | Per-unit retail price |

### Savings

| Property | Description |
|---|---|
| `package.savingsAmount` | Total savings vs retail |
| `package.savingsPercentage` | Savings percentage |
| `package.unitSavings` | Per-unit savings |
| `package.unitSavingsPercentage` | Per-unit savings % |
| `package.hasSavings` | `"true"` if sale price < retail |

### After coupon

| Property | Description |
|---|---|
| `package.discountedPrice` | Unit price after coupon |
| `package.discountedPriceTotal` | Total price after coupon |
| `package.discountAmount` | Coupon discount amount |
| `package.hasDiscount` | `"true"` if coupon active |
| `package.finalPrice` | Alias for `discountedPrice` |
| `package.finalPriceTotal` | Alias for `discountedPriceTotal` |
| `package.totalSavingsAmount` | Retail savings + coupon combined |
| `package.totalSavingsPercentage` | Combined savings % |

### Boolean flags

| Property | Description |
|---|---|
| `package.isBundle` | Multi-unit bundle |
| `package.isRecurring` | Subscription package |
| `package.hasRetailPrice` | Retail price is set |

### Campaign data

| Property | Description |
|---|---|
| `package.campaign.name` | Campaign name |
| `package.campaign.currency` | Campaign currency code |

## Examples

### Price card

```html
<div class="price-card" data-next-package-id="10">
  <span class="badge">Most Popular</span>
  <h3 data-next-display="package.name"></h3>
  <del data-next-display="package.price_retail"></del>
  <p class="price" data-next-display="package.price"></p>
  <small>Save <span data-next-display="package.savingsAmount"></span></small>
</div>
```

### Per-unit pricing

```html
<div data-next-package-id="20">
  <p>3 Bottles — <span data-next-display="package.price"></span></p>
  <small>Only <span data-next-display="package.unitPrice"></span> per bottle</small>
</div>
```

### After-coupon price

```html
<div data-next-package-id="10">
  <del data-next-display="package.price"></del>
  <!-- Updates when a coupon is applied -->
  <strong data-next-display="package.finalPrice"></strong>
</div>
```

---

## Raw values

```html
<!-- Formatted -->
<span data-next-display="package.price"></span>           <!-- $29.00 -->

<!-- Raw number -->
<span data-next-display="package.price.raw"></span>       <!-- 29 -->
```

---
title: Conditional Display
description: Show or hide elements based on cart state, order data, URL parameters, or active profile.
---

`ConditionalDisplayEnhancer` handles `data-next-show` and `data-next-hide` attributes. The element is shown when the condition is truthy (`data-next-show`) or hidden when truthy (`data-next-hide`).

---

## Basic usage

```html
<!-- Show when cart has items -->
<div data-next-show="cart.hasItems">
  <a href="/checkout">Proceed to Checkout</a>
</div>

<!-- Hide when cart is empty -->
<div data-next-hide="cart.isEmpty">
  <p>Cart Summary</p>
</div>
```

---

## Condition syntax

### Property access

```html
<div data-next-show="cart.hasItems">...</div>
<div data-next-show="order.hasSavings">...</div>
```

### Comparison operators

```html
<div data-next-show="cart.total > 100">Free shipping!</div>
<div data-next-show="cart.totalQuantity >= 3">Bundle discount applied</div>
<div data-next-show="order.status == complete">Order is complete</div>
<div data-next-show="cart.itemCount != 0">Cart is not empty</div>
```

### Function calls

```html
<!-- Show if package 123 is in the cart -->
<div data-next-show="cart.hasItem(123)">You selected the starter pack</div>
```

### Negation

```html
<div data-next-show="!cart.isEmpty">Cart is not empty</div>
```

### Logical operators

```html
<!-- AND -->
<div data-next-show="cart.hasItems && cart.total >= 50">
  You qualify for a discount!
</div>

<!-- OR -->
<div data-next-show="cart.isEmpty || param.preview == true">
  Show empty state or preview
</div>
```

### URL parameters

```html
<!-- Show only on mobile traffic -->
<div data-next-show="param.utm_source == mobile">Mobile offer</div>

<!-- Hide in preview mode -->
<div data-next-hide="param.preview == true">Real content</div>
```

---

## Context objects

| Object | Available properties |
|---|---|
| `cart` | `hasItems`, `isEmpty`, `total`, `itemCount`, `totalQuantity`, `savings`, `hasItem(id)` |
| `package` | Package data properties (requires `data-next-selector-id`) |
| `order` | `hasSavings`, `hasItems`, `hasUpsells`, `status`, etc. |
| `shipping` | Shipping method data |
| `profile` | `active`, `id` |
| `param` / `params` | URL search parameters |

---

## Profile-based visibility

Show or hide content based on the active pricing profile:

```html
<!-- Show only for wholesale profile -->
<div data-next-show-if-profile="wholesale">
  <p>Wholesale price: ...</p>
</div>

<!-- Show for everyone except wholesale -->
<div data-next-hide-if-profile="wholesale">
  <p>Retail price: ...</p>
</div>
```

---

## Package conditions with a selector

When using `package.*` conditions, specify the selector ID to track:

```html
<div data-next-show="package.isBundle"
     data-next-selector-id="main">
  Bundle deal — free shipping included!
</div>
```

## Examples

### Checkout button (show when cart has items)

```html
<div data-next-show="cart.hasItems">
  <a href="/checkout" class="btn-primary">
    Checkout — <span data-next-display="cart.total"></span>
  </a>
</div>

<div data-next-hide="cart.hasItems">
  <p>Add a package to get started.</p>
</div>
```

### Free shipping threshold

```html
<div data-next-show="cart.total >= 99">
  🎉 You qualify for free shipping!
</div>

<div data-next-hide="cart.total >= 99">
  Add <span data-next-display="cart.total"></span> more for free shipping
</div>
```

### Quantity-based messaging

```html
<div data-next-show="cart.totalQuantity == 1">Buy 2 more and save 20%</div>
<div data-next-show="cart.totalQuantity == 2">Add 1 more and save 20%</div>
<div data-next-show="cart.totalQuantity >= 3">🎉 Discount applied!</div>
```

### Order status

```html
<div data-next-show="order.hasUpsells">
  Your order has been upgraded with extra items.
</div>
```

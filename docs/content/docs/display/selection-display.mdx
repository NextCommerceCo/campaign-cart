---
title: Selection Display
description: Show data for the currently selected package in a PackageSelectorEnhancer. Updates in real-time as selection changes.
---

`SelectionDisplayEnhancer` handles `data-next-display="selection.<selectorId>.*"` elements. It listens to the named `PackageSelectorEnhancer` and updates whenever the user selects a different card.

This is the best way to show a dynamic price or total that changes as the user picks different packages.

---

## Usage

Embed the selector ID in the display path:

```html
<!-- Selector with id="main" -->
<div data-next-cart-selector data-next-selector-id="main" data-next-selection-mode="select">
  <div data-next-selector-card data-next-package-id="10" data-next-selected="true">1 Bottle</div>
  <div data-next-selector-card data-next-package-id="20">3 Bottles</div>
</div>

<!-- Display the selected card's data -->
<p>Your total: <span data-next-display="selection.main.total"></span></p>
<p>You save: <span data-next-display="selection.main.savingsAmount"></span></p>
```

Alternatively, use the `data-next-selector-id` attribute:

```html
<span data-next-display="selection.price" data-next-selector-id="main"></span>
```

---

## Available properties

### State

| Property | Description |
|---|---|
| `selection.<id>.hasSelection` | `"true"` if a card is selected |
| `selection.<id>.packageId` | Selected package ID |

### Product info

| Property | Description |
|---|---|
| `selection.<id>.name` | Package name |
| `selection.<id>.quantity` | Cart quantity of the selected card |
| `selection.<id>.totalUnits` | Units in the package (qty field) |

### Pricing

| Property | Description |
|---|---|
| `selection.<id>.price` | Package sale price |
| `selection.<id>.total` | price × quantity |
| `selection.<id>.price_total` | Alias for total |
| `selection.<id>.unitPrice` | Per-unit price |
| `selection.<id>.pricePerUnit` | Alias for unitPrice |
| `selection.<id>.compareTotal` | Retail price × quantity |
| `selection.<id>.price_retail_total` | Alias for compareTotal |

### Savings

| Property | Description |
|---|---|
| `selection.<id>.savingsAmount` | Savings vs retail |
| `selection.<id>.savings` | Alias |
| `selection.<id>.savingsPercentage` | Savings % |
| `selection.<id>.hasSavings` | `"true"` if there are savings |

### After coupon

| Property | Description |
|---|---|
| `selection.<id>.discountedPrice` | Price after coupon |
| `selection.<id>.finalPrice` | Alias |
| `selection.<id>.appliedDiscountAmount` | Coupon discount |
| `selection.<id>.hasDiscount` | `"true"` if coupon active |
| `selection.<id>.discountPercentage` | Coupon % |

### Bundle flags

| Property | Description |
|---|---|
| `selection.<id>.isBundle` | `"true"` for multi-unit packages |
| `selection.<id>.isMultiPack` | Alias |

---

## Hiding when nothing is selected

Display elements hide themselves automatically when there is no selection — **except** for `hasSelection`, which always renders.

---

## Math expressions

You can perform inline arithmetic on numeric properties:

```html
<!-- 10% of the total -->
<span data-next-display="selection.main.total*0.1"></span>

<!-- Add shipping -->
<span data-next-display="selection.main.total+4.99"></span>
```

## Examples

### Dynamic order summary

```html
<div id="order-summary">
  <p>Package: <span data-next-display="selection.main.name"></span></p>
  <p>
    <del data-next-display="selection.main.compareTotal"></del>
    <strong data-next-display="selection.main.total"></strong>
  </p>
  <p>You save: <span data-next-display="selection.main.savingsAmount"></span>
     (<span data-next-display="selection.main.savingsPercentage"></span>)</p>
</div>
```

### Per-unit callout

```html
<!-- Shows when a bundle is selected -->
<p data-next-show="selection.main.isBundle">
  Only <span data-next-display="selection.main.unitPrice"></span> per bottle!
</p>
```

### Final price with coupon

```html
<p>
  <del data-next-display="selection.main.total"></del>
  <strong data-next-display="selection.main.discountedPrice"></strong>
</p>
```

title: Display System
description: Reactive data display — prices, totals, order data, conditional visibility, and timers.
---

Display enhancers render live data into HTML elements and update automatically when the underlying store changes. No JavaScript needed.

The core attribute is `data-next-display` with a dot-separated path:

```html
<span data-next-display="cart.total"></span>
<span data-next-display="package.price"></span>
<span data-next-display="selection.main.total"></span>
<span data-next-display="order.refId"></span>
```

The **first segment** determines which data source (and enhancer) handles the element:

| Prefix | Enhancer | Data source |
|---|---|---|
| `cart.*` | `CartDisplayEnhancer` | Cart store — totals, item count |
| `package.*` | `ProductDisplayEnhancer` | Campaign store — package prices and info |
| `selection.<id>.*` | `SelectionDisplayEnhancer` | Currently selected card in a `PackageSelectorEnhancer` |
| `order.*` | `OrderDisplayEnhancer` | Order store — post-purchase data |
| `shipping.*` | `ShippingDisplayEnhancer` | Cart store — shipping methods |

## Enhancers in this section

| Enhancer | Attribute | Purpose |
|---|---|---|
| [`CartDisplayEnhancer`](/docs/display/cart-display) | `data-next-display="cart.*"` | Cart totals, item count, empty state |
| [`ProductDisplayEnhancer`](/docs/display/product-display) | `data-next-display="package.*"` | Package prices, savings, variant data |
| [`SelectionDisplayEnhancer`](/docs/display/selection-display) | `data-next-display="selection.<id>.*"` | Data for the selected card |
| [`OrderDisplayEnhancer`](/docs/display/order-display) | `data-next-display="order.*"` | Post-purchase order data |
| [`ConditionalDisplayEnhancer`](/docs/display/conditional-display) | `data-next-show` / `data-next-hide` | Show/hide based on conditions |
| [`TimerEnhancer`](/docs/display/timer) | `data-next-timer` | Countdown timer with persistence |
| [`FormatCurrencyEnhancer`](/docs/display/format-currency) | `data-next-format-currency` | Format a static number as currency |

## Format control

By default, prices are formatted as currency and percentages include a `%` sign. Override with `data-next-format`:

| Format | Result |
|---|---|
| `currency` (default for prices) | `$29.00` |
| `percentage` | `15%` |
| `number` | `29` |
| `text` | Plain string |
| `boolean` | `"true"` / `"false"` |
| `raw` | Unformatted number |

```html
<!-- Formatted (default) -->
<span data-next-display="cart.total"></span>          <!-- $49.00 -->

<!-- Raw number -->
<span data-next-display="cart.total.raw"></span>      <!-- 49 -->
<span data-next-display="cart.total" data-next-format="raw"></span>
```

## Common display patterns

```html
<!-- Cart summary bar -->
<span data-next-display="cart.itemCount"></span> items —
<span data-next-display="cart.total"></span>

<!-- Show savings -->
You save <span data-next-display="cart.savings"></span>
(<span data-next-display="cart.savingsPercentage"></span> off)

<!-- Package price with strikethrough -->
<del data-next-display="package.price_retail"></del>
<strong data-next-display="package.price"></strong>

<!-- Selected item total -->
Order total: <span data-next-display="selection.main.total"></span>

<!-- Order confirmation -->
Thank you <span data-next-display="order.user.firstName"></span>!
Your order <span data-next-display="order.refId"></span> is confirmed.
```
